about this site

intro

The Herbivore Handbook is a cozy vegan resource site for ethics, politics, practical questions, recipes, links, and personal commentary.

It is meant to be easy to browse: start with veganism 101, read the case for veganism and FAQ, then use the other sections for deeper political context, recipes, and further reading.
